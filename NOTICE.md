## Maccy-Derived macOS UI

`copyagent-ui-mac` is derived from Maccy, an open-source macOS clipboard manager by Alex Rodionov. The original Maccy MIT license notice is preserved in `copyagent-ui-mac/LICENSE` and must remain with source and binary distributions that include the derived UI.

## Third-Party Dependencies

- Go dependencies are listed in `go-copyagentd/go.mod` and `go-copyagentd/go.sum`.
- Node prototype dependencies are listed in `package.json` and `package-lock.json`.
- Swift Package dependencies are resolved through `copyagent-ui-mac/Copyagent.xcodeproj/project.xcworkspace/xcshareddata/swiftpm/Package.resolved`.

Review dependency licenses before publishing binary bundles or package-manager artifacts.
