## 0.1.0-alpha - 2026-04-26

Initial public alpha release shape:

- `copyagentd` Go daemon for Feishu/Lark transport, deterministic actions, diagnostics, and LaunchAgent lifecycle
- optional `copyagent-ui-mac` companion for local macOS clipboard UX
- stable Direct Mode for text copy, image/file handling, and fixed replies
- experimental Agent Mode and experimental foreground-hosting commands
- first public setup, permissions, diagnostics, contribution, and security documentation
