# copyagent icon concepts

内部设计说明，面向图标与品牌迭代，不属于公开用户使用文档。

根据项目文档提炼的关键词：移动端聊天、桌面剪切板、本机轻量 agent、文件落地、macOS 菜单栏。

- `01 手机 → 剪切板`：最直接表达“手机消息进入桌面剪切板”。
- `02 本机 Agent`：强调 copyagent 是轻量机器人/路由器。
- `03 聊天命令`：强调飞书/移动端聊天指令入口。
- `04 文件落地`：兼顾图片、文件保存到本机的能力。
- `05 同步循环`：更像 macOS 状态栏/工具图标，抽象且耐看。
